Let me demonstrate the Variational Principe by using it with the simple PIB system. It will be a nice comparison because we already know what the answer should be.

What the Variaional Principle gives us is an estimate (for the upper bound) for the first energy level of a system

Lets first write down the Hamiltonian for the PIB system. \[ \hat{H} = -\frac{\hbar^{2}}{2m} \nabla^{2} + 0 \]

Lets pretend we do not know the wave function. Lets use a trial wave function that satisfies the boundary conditions: \[ f = {\color{myGreen}A x (L-x)} \]

Getting ready: \[ \h f=\h {\color{myGreen}A x (L-x)}= -\frac{\hbar^{2}}{2m} \nabla^{2} {\color{myGreen}A x (L-x)}= -\frac{\hbar^{2}}{2m}\daa{{\color{myGreen}A x (L-x)}}{x} = \frac{\hbar^{2}}{m}A \]

Using the variational principle: \[\begin{align} w&=\frac{\displaystyle \int f^{*} \h {f} \,dx}{\displaystyle \int f^{*}f \,dx} = \frac{\displaystyle \int \left[ {\color{myGreen}A x (L-x)}\right]^{*}\left(\frac{\hbar^{2}}{m}A\right) \,dx}{\displaystyle \int \left[ {\color{myGreen}A x (L-x)}\right]^{*} [{\color{myGreen}A x (L-x)}]\,dx} = \frac{5 \hbar^{2}}{m L^{2}} = \frac{10}{\pi^{2}}\left( \frac{\pi^{2} \hbar^{2}}{2 m L^{2}}\right) =1.01321 E_{1} \end{align}\]
